Element's Name

Web components must contain at least one dash in its name.

failure hello-digitas -bug top-10 4-the-money

validator

Web Components Standards approved!

- It can not be the first letter - It should not end with the dash - Avoid the use of vendor prefixes - youtube gaming example

Life Cycle

  • created || createdCallback
  • ready
  • factoryImpl
  • attached || attachedCallback
  • attributeChanged || attributeChangedCallback
  • detached || detachedCallback

Order:

  1. created callback
  2. local DOM initialized
  3. ready callback
  4. factoryImpl callback
  5. attached callback
