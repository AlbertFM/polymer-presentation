Element's Name

Web components must contain at least one dash in its name.

x-element (not recomended)

Web Components Standards approved!

- It can not be the first letter - It should not end with the dash - Avoid the use of vendor prefixes - youtube gaming example

Life Cycle

  • created || createdCallback
  • ready
  • factoryImpl
  • attached || attachedCallback
  • attributeChanged || attributeChangedCallback
  • detached || detachedCallback

Order:

  1. created callback
  2. local DOM initialized
  3. ready callback
  4. factoryImpl callback
  5. attached callback

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Take a Moment

Press B or . on your keyboard to pause the presentation. This is helpful when you're on stage and want to take distracting slides off the screen.
